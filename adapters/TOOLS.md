# SEO Tool Connection Catalog

Use this list to decide which tools should feed the adapter layer. Prefer the free baseline first, then add paid tools only when site size, revenue, reporting needs or risk justifies them.

## Free First-Party Search Data

- Google Search Console: indexing, search performance, sitemaps, enhancements and security signals.
- Bing Webmaster Tools: Bing indexing, crawl diagnostics, backlinks and sitemap data.
- PageSpeed Insights: Lighthouse and field/lab performance diagnostics.
- CrUX API or CrUX exports: field Core Web Vitals by URL or origin.
- Rich Results Test: structured data eligibility checks.
- Google Business Profile: local profile, reviews, categories, hours and location data.

## Crawlers and Technical Audit Tools

- Screaming Frog SEO Spider: desktop crawler, JavaScript rendering, custom extraction, sitemap generation and technical exports.
- Sitebulb: visual crawl reporting and technical audit prioritization.
- Lumar: enterprise crawl monitoring and large-site governance.
- Botify: enterprise crawl, logs and search performance analysis.
- Oncrawl: crawl and log-file analysis.

## Analytics and Behavior

- GA4: landing page, engagement and conversion data.
- Looker Studio: reporting views from GSC, GA4 and paid platforms.
- Microsoft Clarity: heatmaps and session recordings.
- Hotjar: behavior feedback and recordings.

## Rank, SERP and Keyword Data

- DataForSEO: API-first keyword, SERP, rank, local, marketplace and review data.
- Semrush: keyword, competitor, rank, backlink, content and AI visibility data.
- Ahrefs: backlinks, keywords, content gaps, rank tracking and competitive research.
- SE Ranking: rank tracking, audits, competitor research and AI visibility features.
- STAT: enterprise SERP/rank tracking.
- AccuRanker: rank tracking and SERP features.

## Backlinks, Digital PR and Mentions

- Ahrefs: backlink index and competitive link analysis.
- Majestic: backlink graph and topical trust signals.
- Semrush Backlink Analytics: link discovery and monitoring.
- BuzzStream: outreach and relationship tracking.
- Respona: outreach workflows.
- Brand24, Mention or Talkwalker: brand mention monitoring.

## Local SEO

- Google Business Profile API: location, category, hours, photo, post and review workflows.
- BrightLocal: local rank tracking, citations and local audit reports.
- Whitespark: citation discovery and local rank tracking.
- Yext: enterprise listings management.
- Moz Local: listing distribution and consistency.

## Accessibility, Rendering and Quality

- axe DevTools or axe-core: accessibility checks.
- Lighthouse: accessibility, performance, SEO and best-practice audits.
- Playwright: rendered DOM and user-flow testing.
- WAVE: accessibility review support.

## AI Search, GEO and Entity Monitoring

- Perplexity, ChatGPT Search, Google AI Overviews and Gemini snapshots: AI citation monitoring where terms permit.
- AlsoAsked or similar question tools: conversational query discovery.
- Wikidata, Google Knowledge Graph-style entity sources and organization profiles: entity consistency checks.
- DataForSEO SERP APIs: AI/organic visibility monitoring when supported.

## Server Logs, Security and Compliance

- Server/CDN logs: Googlebot, Bingbot and AI crawler access.
- Cloudflare, Akamai or Fastly logs: bot, cache and security behavior.
- Google Safe Browsing and GSC Security Issues: hacked page and malware signals.
- Privacy/cookie platforms: consent and disclosure checks.

Sources checked for tool categories and API patterns include official pages for Screaming Frog SEO Spider, Ahrefs, DataForSEO and Google Search Console API authorization.
