# Examples

These examples show complete, filled-in outputs using the repository's agents, skills, templates, and schemas.

## Included Examples

- `full-audit-example/`: sample full SEO audit output.
- `content-brief-example/`: sample content brief output.
- `technical-deployment-example/`: sample technical SEO implementation plan and handoff.
- `anonymized-production-style-example/`: crawl, GSC-style and GA4-style inputs with a matching agent output.
- `anonymized-real-site-gsc-export/`: sanitized GSC export pattern for real-site data.
- `schema-validation-examples/`: JSON examples validated against repository schemas.

The examples use fictional, synthetic or anonymized/sanitized data. They demonstrate structure and validation patterns, not live public SEO findings.
