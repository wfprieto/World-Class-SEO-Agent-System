# SEO Diagnostic Infrastructure Plan: small local service business

Lead agent: SEO Diagnostic Infrastructure Agent

## Budget and Context

| Question | Answer |
| --- | --- |
| Monthly SEO tool budget | $0 |
| Website size | 1-20 pages |
| Business type | Local business |
| Website or app platform | WordPress |
| Code and deployment access | CMS/admin access only |
| Technical ability | Comfortable with CMS/plugins |
| Reporting need | Monthly owner summary |
| Existing tools | None confirmed |

## Recommended Stack

| Priority | Tool or System | Free/Paid | Purpose | Owner | Setup Effort |
| --- | --- | --- | --- | --- | --- |
| P0 | Google Search Console | Free | Search visibility and indexing diagnostics | Site owner or SEO operator | Low |
| P0 | Bing Webmaster Tools | Free | Additional search engine diagnostics | Site owner or SEO operator | Low |
| P0 | GA4 | Free | Traffic and conversion measurement | Site owner or developer | Medium |
| P1 | Looker Studio | Free | Simple monthly dashboard | SEO operator | Medium |
| P1 | PageSpeed Insights | Free | Performance and Core Web Vitals checks | Developer or SEO operator | Low |
| P2 | Screaming Frog free mode | Free | Small-site crawl checks up to free limit | SEO operator | Low |

## Free Baseline

Start with free search-engine, analytics, performance, and crawl diagnostics. This is enough for a small local site before buying paid SEO tools.

## Paid Upgrade Path

Do not buy paid tools yet. Consider a low-cost rank tracker or paid crawler only after the free setup is working and the site needs recurring reports.

## Diagnostic Setup Checklist

- Verify Google Search Console.
- Verify Bing Webmaster Tools.
- Install GA4 and define lead/contact conversion events.
- Create a simple Looker Studio dashboard.
- Run a baseline crawl and PageSpeed check.

## SEO Grading Model

Use a simplified 100-point score covering technical setup, local presence, content clarity, conversion tracking, and review/citation readiness.

## Reporting Dashboard Plan

Create a one-page dashboard showing search clicks, top pages, top queries, conversions, technical warnings, and next actions.

## Monthly Reporting Cadence

Review once per month and after any website launch or major content update.

## Data Quality Checks

| Data Source | Quality Check | Failure Signal | Owner |
| --- | --- | --- | --- |
| GSC | Property verified and sitemap submitted | No data after setup period | SEO operator |
| GA4 | Contact form conversion tracked | No conversions firing after test submission | Developer |

## Alert Thresholds

Escalate if search clicks drop sharply, pages fall out of index unexpectedly, or conversion tracking stops working.

## Plain-Language Stakeholder Report Plan

Use the SEO Output Report Agent to create a short monthly summary explaining what changed, what improved, what needs attention, and what to do next.

## What Not To Buy Yet

Do not buy enterprise crawlers, expensive backlink tools, or advanced content optimization tools until the free diagnostic baseline is working.

## Next Steps

1. Set up GSC and Bing Webmaster Tools.
2. Install GA4 conversion tracking.
3. Build the first monthly Looker Studio dashboard.
