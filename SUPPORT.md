# Support

## Questions and how-to help

Use [GitHub Discussions](https://github.com/wfprieto/World-Class-SEO-Agent-System/discussions) for usage questions, implementation ideas, and general SEO workflow discussion.

## Bugs and documentation problems

Open a GitHub issue using the appropriate issue form. Please include a minimal, non-sensitive reproduction or a specific documentation location.

## Security issues

Do not include vulnerability details, credentials, client data, or exploit code in a public issue. Follow the private-reporting instructions in [SECURITY.md](SECURITY.md).
