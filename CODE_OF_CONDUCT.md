# Code of Conduct

## Our Standard

Contributors should help make this repository useful, accurate, safe, and respectful.

Examples of positive behavior:

- Improving evidence quality
- Clarifying workflows
- Adding safer implementation guidance
- Respecting different levels of SEO experience
- Challenging ideas with evidence rather than personal criticism

Examples of unacceptable behavior:

- Harassment or personal attacks
- Spam or promotional abuse
- Submitting manipulative SEO tactics as recommendations
- Encouraging fake reviews, fake locations, cloaking, doorway pages, or link schemes
- Publishing private or sensitive information

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block contributors who violate this code of conduct.

## Reporting

Report conduct concerns to the repository owner.
