# Comparative Benchmarks

The benchmark framework measures outcomes, not file counts.

## Archetypes

SaaS, e-commerce, local service, publisher, agency/B2B, multilingual, and JavaScript-heavy sites.

## Required tasks

Full audit, page audit, content brief, technical diagnosis, schema, CWV, local, e-commerce, hreflang, decay, backlinks, GEO/AIO, drift, roadmap, and executive report.

## Hard gates

- Critical/high defect recall >= 95%
- Finding precision >= 90%
- Unsupported factual claims = 0
- Schema-valid outputs = 100%
- Required handoffs consumed = 100%
- Secret exposure = 0
- Unbounded paid calls = 0
- Install and documented-command success = 100%

`evaluation/comparative/final-release-readiness.json` records which gates are proven, blocked, or not run. Missing evidence never becomes a pass.
