# Knowledge Sources

Use official and first-party sources before third-party commentary.

## Primary Search Sources

- Google Search Central: https://developers.google.com/search
- Google Search Status Dashboard: https://status.search.google.com/
- Bing Webmaster Guidelines: https://www.bing.com/webmasters/help/webmaster-guidelines-30fba23a
- Schema.org: https://schema.org/

## Accessibility Sources

- W3C WCAG overview: https://www.w3.org/WAI/standards-guidelines/wcag/
- WCAG 2.2: https://www.w3.org/TR/WCAG22/

## Data Sources

Use when available:

- Google Search Console
- Bing Webmaster Tools
- Web analytics
- Server logs
- CRM or revenue data
- Rank tracking
- Crawl exports
- PageSpeed or CrUX data
- Backlink data
- Local citation and review platforms

## Source Rules

- Official documentation can create rules.
- First-party data can override generic advice.
- Experiments can create local rules after review.
- Industry commentary can create hypotheses, not mandatory rules.
- Every rule update must include confidence and review date.
